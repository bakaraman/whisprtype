# Permissions

WhisprType needs two macOS permissions for the full product experience.

## Microphone

Needed for speech capture.

Open:

```text
System Settings → Privacy & Security → Microphone
```

## Accessibility

Needed for paste/type automation into other apps.

Open:

```text
System Settings → Privacy & Security → Accessibility
```

## Notes

- Clipboard-only mode can reduce automation requirements, but the intended experience still benefits from Accessibility access.
- If paste is failing, check Accessibility first.
- If recording is silent, check Microphone first.
